唯一索引

创建唯一索引

方法一

格式：create unique index 索引名 on 表名(字段名)；

方法二

格式：alter table 表格名称 add unique(字段名称)；

删除唯一索引

格式：alter table 表格名称 drop index 字段名;

外键

外键：一个表的某个字段指向另个表的主键，称为外键。 主表（父表）：被指向的表 从表（子表）：主键所在表
